KPI Card Pro displays your most important business metrics in a single, 
clean tile — current value, prior-period comparison, target tracking, 
variance indicator and a compact sparkline trend chart.

Designed for executive dashboards that need information density without 
visual clutter.

KEY FEATURES
- Current value with auto display units (K, M, B)
- Prior period variance with trend arrow
- Target tracking with goal line in sparkline
- Compact sparkline (line, area or bar)
- Conditional color formatting (positive/negative/neutral)
- High contrast mode for accessibility
- Full formatting pane parity in Desktop and Service

FREE TIER
- 1 KPI metric with sparkline and variance

PRO TIER (via AppSource)
- Sparkline type selector (line, area, bar)
- Variance pill with custom colors
- Target line in sparkline
- Tooltip drill-through with custom fields
- Custom prefix, suffix and card styling
