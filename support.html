Getting Started

KPI Card Pro requires at least one measure in the Value field well. Add an optional measure to Prior Period to enable variance calculation, and a date/time dimension to Category to enable the sparkline trend chart.

Field Wells

  • Value — Current period KPI value (required)
  • Prior Period — Previous period value for variance (optional)
  • Target — Goal or budget value (optional, Pro)
  • Category — Time dimension for sparkline (optional)
  • Tooltips — Additional drill-through fields (optional, Pro)

Free vs Pro

  • Free: 1 KPI metric, basic sparkline, fixed colors, variance text
  • Pro: Sparkline type selector, variance pill, custom colors, target line, tooltip drill-through, custom prefix/suffix and card styling

Upgrade to Pro via Microsoft AppSource.

Frequently Asked Questions

The sparkline is not showing — why?
The sparkline requires a field in the Category well (e.g. Month, Quarter, Year). Without it, there is only one data point and no trend to draw.

The variance shows the same value for both periods.
Make sure your Prior Period measure filters a different time period than your Value measure. Use CALCULATE(..., tabla[Year] = MAX(tabla[Year]) - 1) as a starting point.

I upgraded to Pro but features are still locked.
Sign out of Power BI Service and sign back in to refresh your license. If the issue persists, contact us at the email below.

The visual is not rendering in Power BI Desktop.
Make sure you are using Power BI Desktop April 2024 or later. Re-import the .pbiviz file via Visualizations → … → Import a visual from a file.
